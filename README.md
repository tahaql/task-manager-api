# 📝 Task Manager API

A simple RESTful API for managing tasks, built with Node.js, Express, and MongoDB.

## 🚀 Features
- Create, read, update, and delete tasks (CRUD)
- Mark tasks as complete/incomplete
- Timestamp for creation and updates
- RESTful API design

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
